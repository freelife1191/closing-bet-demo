## ✅ Priority 1: 장중 실시간 수급 데이터 연동 (KIS API)

현재 시스템은 장 마감 후의 수급 데이터만 반영하고 있어, 장중에는 수급 점수가 0점으로 고정되는 한계가 있습니다. 이를 해결하기 위해 **한국투자증권(KIS) Open API** 연동 준비를 마쳤으며, 계좌 개설 및 키 발급 후 즉시 연동을 재개할 수 있습니다.

### 📌 작업 목록 (Checklist)
- [ ] **한국투자증권 계좌 개설** (비대면 가능)
- [ ] **Open API 서비스 신청** (HTS 또는 홈페이지)
- [ ] **API Key 발급** (App Key, App Secret 확보)
- [ ] **`.env` 파일 설정**
    - `KIS_APP_KEY`, `KIS_APP_SECRET`, `KIS_ACCOUNT_NO` 입력
- [ ] **연동 활성화 요청**
    - "KIS API 키 설정했어, 연동해줘"라고 AI에게 요청
