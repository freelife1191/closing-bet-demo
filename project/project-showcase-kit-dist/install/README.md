# Installer Entry

Use one command to install canonical `psk-*` skills into root `.agent/skills`.

```bash
bash project/project-showcase-kit-dist/install/install_all.sh codex
bash project/project-showcase-kit-dist/install/install_all.sh claudecode
bash project/project-showcase-kit-dist/install/install_all.sh gemini
bash project/project-showcase-kit-dist/install/install_all.sh antigravity
```
